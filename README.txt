Como publicar:

2) No GitHub do site, substitua o arquivo 'wingchun.html' na raiz.
3) Envie os 3 arquivos para a pasta 'foto/' do repositório (crie a pasta se não existir).
4) Confirme o commit.
5) Abra https://ivaldo25psicanalista.github.io/wingchun.html?v=3
